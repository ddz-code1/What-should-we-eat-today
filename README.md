# 今天吃什么・食材菜谱助手

一个简洁清新的食材菜谱匹配工具，支持移动端和PC端，采用扁平化设计风格。

## ✨ 功能特色

### 🏠 首页 - 智能菜谱匹配
- **食材选择**：支持蔬菜、肉类、主食分类选择，最多10种食材
- **厨具匹配**：一键选择烹饪工具（炒锅、烤箱、微波炉等）
- **智能匹配**：三种匹配模式
  - 模糊匹配：包含任意选中食材
  - 严格匹配：必须包含所有选中食材
  - 生存模式：简单快速的极简菜谱

### 🎲 吃什么 - 随机菜谱生成
- **数量控制**：1-20道菜谱随机生成
- **一键随机**：点击"随机一下"立即生成
- **菜谱详情**：点击查看完整制作步骤

### 👤 我的 - 个人管理中心
- **历史记录**：查看生成过的菜谱
- **我的收藏**：收藏喜欢的菜谱
- **自定义菜谱**：创建个性化菜谱
- **食材保质期管理**：智能过期提醒系统

## 🍎 食材保质期管理

### 核心功能
- **添加食材**：记录食材名称、购买时间、保质期
- **智能提醒**：
  - 提前3天提醒即将过期
  - 提前1天高频提醒
  - 过期当天强制提醒
- **批量管理**：支持批量删除、标记已食用
- **状态显示**：按剩余天数智能排序和标色

### 提醒机制
- **页面提醒**：底部导航栏显示提醒徽章
- **弹窗提醒**：过期食材强制弹窗
- **Toast提示**：轻量级过期提醒

## 🚀 快速开始

1. **启动本地服务器**
   ```bash
   # Python 3
   python -m http.server 8000

   # 或使用其他HTTP服务器
   ```

2. **打开浏览器**
   ```
   http://localhost:8000
   ```

3. **开始使用**
   - 在首页选择食材和厨具
   - 点击生成菜谱
   - 或切换到"吃什么"页面随机生成
